Detailed Description

Implementation of the class "TSQueue".

Implementation of a thread-safe queue with fine granular locks. The implementation was borrowed from the book: Anthony Williams, C++ Concurrency in Action, 2012, ISBN: 978-5-97060-194-5 https://dmkpress.com/catalog/computer/programming/c/978-5-94074-448-1/

The implementation is slightly modified to the requirements of the program.
